-------------------
- PASS: Release build against the official Vintage Story 1.22.3 server files
  completed with 0 errors (5 pre-existing nullable-analysis warnings).
- PASS: The package contains exactly nine enabled Apprentice race definitions.
- PASS: No assets/game/config/characterclasses.json shadow override is packaged.

Required in-game test
---------------------
1. Back up the world and install only Apprentice 2.2.1d.
2. Remove/disable every older Apprentice ZIP and unpacked Apprentice mod path.
3. Join with a player whose previous class came from vanilla or another mod.
4. Confirm the server logs one Apprentice legacy-class reset notification.
5. Confirm race selection opens automatically without entering `.charsel`.
6. Select a race, reconnect, and confirm the chosen race remains selected.
7. Run `/player <name> allowcharselonce`, then `.charsel`, and confirm it opens.
8. Confirm the Apprentice mastery/skill-tree GUI retains the 2.1.14i layout.
