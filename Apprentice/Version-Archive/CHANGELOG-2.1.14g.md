# Apprentice 2.1.14g — Skill-Tree Spacing Polish

## Interface

- Raised the `path-a`, `path-b`, and capstone node labels so names such as
  **Duelist**, **Berserker**, and **Warrior Grandmaster** no longer collide
  with connector lines or the bottom of the Skill Tree panel.
- Added a compact panel-colored backing behind node labels so connector lines
  cannot show through their glyphs.
- Centered every profession name within its sidebar row and increased its
  preferred font scale while retaining the profession icon and level counter.
- Removed extra composer padding around the fixed Apprentice canvas so the
  interface consumes the broad brown area at the right and bottom. The shaded
  dialog artwork still supplies the visible frame.
- Fitted status messages to the status bar so long text does not extend into
  the surrounding panel.
- Kept the fixed `1280 x 780` Apprentice content size, compact tabs, and the
  **HIDDEN** tab label.

## Compatibility

- Existing saves remain compatible.
- Tree IDs, node IDs, hidden-class IDs, and tab IDs are unchanged.
