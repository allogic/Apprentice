# Apprentice 2.1.14i — Centered Profession Levels

## Interface

- Centered every `Lv N` label horizontally in the rightmost 58-unit column of
  its profession row.
- Lowered the smaller level-label baseline by two design units so it shares
  the same visual vertical center as the profession name.
- Profession names, icons, row bounds, and hit regions are unchanged.
- All 2.1.14h layout refinements are retained.

## Compatibility

- Existing saves remain compatible.
- Tree IDs, node IDs, hidden-class IDs, and tab IDs are unchanged.
