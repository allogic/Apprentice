# Apprentice 2.1.14a — Crosshair Hotfix

## Based on 2.1.14

This hotfix keeps the 2.1.14 Ranger / movement rebalance changes, but restores the preferred icon treatment for two Ranger nodes and cleans up requirement marker alignment.

## New in 2.1.14a

- Restored the old **crosshair-style icon** for **Sharpshooter**.
- Restored the old **crosshair-style icon** for **Master Sharpshooter**.
  - Master Sharpshooter now uses an enhanced crosshair with a larger outer ring and center point.
- Adjusted the requirement markers so the **"+"** and **"-"** symbols sit more centrally inside their circular indicators.

## Compatibility

- Existing save data remains compatible.
- Tree IDs and node IDs are unchanged.
- No gameplay effects were altered in this hotfix.
