# Apprentice 2.1.10 — Skill Tree Clarity Update

## Based on 2.1.9

This release keeps the 2.1.7 purchase-flow fixes, the 2.1.8 icon foundation, and the 2.1.9 skill-tree icon overhaul.

## New in 2.1.10

- Improved the **node effects text** so it now explains the change more clearly:
  - shows the affected stat name
  - shows the **current bonus**
  - shows the **next-rank bonus**
- Reworded many node effect labels so bonuses are clearer and less placeholder-like.
  - Example: vague labels like `Skirmisher bonus` and `Arrowstorm bonus` were replaced with clearer stat-oriented text such as **Mobile ranged damage** and **Volley damage**.
- Reworked the **Hunter profession icon** to remove the awkward look from the previous version.
- Reworked the **first skill in every tree** so it is no longer just a generic experience boost.
  - Each foundation node now has a more individual theme, such as:
    - Sure Swing
    - Lumber Grip
    - Green Thumb
    - Steady Hand
    - Forge Rhythm
    - True Aim
    - Eagle Eye
    - Long Reach
    - Shield Drill
    - Iron Constitution
- Added a small gameplay-side improvement for the **Hunter** tree by assigning hunting weapon patterns so Hunter weapon-damage bonuses can apply to common hunting weapons.
- Added **Shield** foundation health support so its new maximum-health effect contributes to player stats.
- Removed three clutter / guide text lines from the GUI to clean up the interface.

## Compatibility

- Existing save data remains compatible.
- Node IDs are unchanged.
- No external icon textures are required.
