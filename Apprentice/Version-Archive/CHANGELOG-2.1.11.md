# Apprentice 2.1.11 — Header XP Counter and Tree Cleanup

## Based on 2.1.10

This release keeps the 2.1.7 purchase-flow fixes, the 2.1.8 icon foundation, the 2.1.9 icon overhaul, and the 2.1.10 node-text / foundation-skill improvements.

## New in 2.1.11

- Added a new **XP counter next to the class XP bar** in the header.
  - The bar was made smaller to create space.
  - The new text shows:
    - the **current total XP** for the selected skill tree
    - the **XP still needed for the next level**
- Removed the confusing highlighted yellow path lines in the skill tree.
  - Connection lines now render only in their normal subtle state or the purchased-state color.
  - The misleading extra highlighted lines shown in the screenshot are gone.
- Keeps the 2.1.10 Hunter icon, clearer node-effect descriptions, and the more individualized foundation skills.

## Compatibility

- Existing save data remains compatible.
- Node IDs are unchanged.
- No external icon textures are required.
